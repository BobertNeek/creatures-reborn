# Expanded Brain Blueprints

The expanded brain path keeps the C3/DS lobe, tract, dendrite, and SVRule substrate. It does not replace creatures with an opaque policy.

Profiles:

- `ClassicC3Ds`: minimum classic interface lobes such as `driv`, `decn`, `verb`, `noun`, and `attn`.
- `RebornExpanded`: a larger inspectable profile with additional perception, place memory, social memory, danger memory, curiosity, goal persistence, inhibition, and rest-regulation lobes.

Blueprints generate normal brain and biochemistry genes for pure sim testing. Imported C3/DS genomes are unaffected unless a modern extension overlay explicitly selects an expanded profile.
